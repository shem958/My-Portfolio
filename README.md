

# **Project Title**

**Shem Gikunda Portfolio** – Full-Stack Developer Personal Website


## **Project Description**

This is the **personal portfolio website** of Shem Gikunda, a Full-Stack Software Engineer. The portfolio showcases professional experience, technical projects, skills, and contact information in a clean, interactive, and modern design. The website is **fully responsive**, optimized for performance, and includes animated elements and smooth scroll interactions.

**Key Features:**

* Hero section with **animated background and floating tech icons**.
* **Responsive navigation** with a mobile menu.
* **Section highlighting skills, projects, and experience**.
* Integration of **GitHub links and contact section**.
* Smooth **scroll animations** for a modern UX.
* **Dark theme** using MUI’s ThemeProvider with customization.
* Fully built with **Next.js 16, React, Material UI, Tailwind CSS, and TypeScript**.


## **Tech Stack**

* **Frontend:** Next.js 16, React.js, Material UI, Tailwind CSS
* **Animations:** Framer Motion, Scroll Animations
* **Icons:** React Icons
* **Backend (optional contact form):** Can be extended with Node.js/NestJS/Golang APIs
* **Styling & Themes:** MUI ThemeProvider, Dark Mode support
* **TypeScript** for type safety and scalability


## **Installation**

1. **Clone the repository**

```bash
git clone https://github.com/shem958/shem-portfolio.git
cd shem-portfolio
```

2. **Install dependencies**

```bash
pnpm install
```

*(or use `npm install` / `yarn install` if you prefer)*

3. **Run the development server**

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

4. **Build for production**

```bash
pnpm build
pnpm start
```
